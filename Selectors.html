Selectors

Selector => Elementlere sitil vermek istediğinizde elementleri bulmak ya da seçmek için kullanılan yapılar

  1. Simple Selectors

    Basit Seçiciler => Name, Id, Class'a göre element seçmenizi sağlar
  2. Combinator Selectors

    Birleştirici Seçiciler => Elementler arasındaki spesifik ilişki durmuna göre element seçmemizi sağlar
  3. Pseudo-class Selectors

    Sözde Sınıf Seçiciler => Mevcut durumuna göre element seçmemizi sağlıyor
  4. Pseudo-elements Selectors

    Sözde Element Seçiciler => Elementi seçip bir parçasına css uygulamamızı sağlar
  5. Attribute Selectors

    Öznitelik Seçiciler => Attribute ya da attribute değerine göre element seçmemizi sağlar
